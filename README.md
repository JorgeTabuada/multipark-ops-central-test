# 🚀 MultiPark Ops Central - TESTE

## 🎯 Objetivo

Este é um repositório de teste para validar o deploy e funcionalidades da aplicação MultiPark Ops Central.

## 🔧 Setup Rápido

```bash
# Instalar dependências
npm install

# Executar local
npm run dev

# Build produção
npm run build
```
